```bash

pip install pygame

# Run the game
python main.py
```

### 🎮 Controls:
- **WASD/Arrow Keys**: Move character
- **E**: Interact with learning stations
- **Space**: Start game / Select in combat
- **Enter**: Confirm actions in combat
- **ESC**: Return to menu / Flee from combat
- **TAB**: Toggle stats panel
- **M**: Toggle minimap
